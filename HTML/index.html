# 什么是HTML?

  • HTML是用来描述网页的一种语言

  • HTML指超文本标记语言(HyperTextMarkupLanguage)

  • HTML不是编程语言，是一种标记语言

# HTML5的新特性

  • 用于绘画的canvas标签
  • 用于媒介回放的video和audio元素
  • 对本地离线储存的史好支持
  • 新的特殊内容元素
  • 如：article、footer、headennav、
  • section
  • 新的表单控件
  • 如：calendar、date、time、email、url、search
  • 浏览器的支持
  • Safari、ChromesFirefox以及Opera包括IE9基本支持了HTML5
